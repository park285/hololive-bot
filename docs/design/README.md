# Design Docs

아직 구현 중이거나 future proposal 성격의 설계 문서를 둡니다.

## 포함 기준
- RFC
- 리팩터 설계
- 구현 전 design spec

## 연결 규칙
- 설계 문서는 승인 후 implementation plan으로 연결합니다.
- 구현이 끝나 현재 SSOT가 되면 `docs/current/`로 승격하거나 현재 문서에서 링크합니다.
- 폐기되면 `docs/history/`로 이동합니다.
