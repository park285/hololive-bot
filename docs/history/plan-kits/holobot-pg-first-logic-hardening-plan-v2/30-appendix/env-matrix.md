# Appendix. Environment Matrix

## Existing mode env

| Env | Default | 의미 |
|---|---:|---|
| `ALARM_DISPATCH_PUBLISH_MODE` | `pg_first` in prod compose | publisher mode |
| `ALARM_DISPATCH_CONSUMER_MODE` | `pg` in prod compose | consumer mode |
| `ALARM_DISPATCH_SHADOW_FATAL` | `false` | shadow insert failure fatal 여부 |
| `ALARM_DISPATCH_WAKEUP_ENABLED` | `true` | wakeup token 사용 여부 |
| `ALARM_DISPATCH_MAX_DELIVERIES_PER_BATCH` | `1000` | publisher insert chunk size |

## Consumer env

| Env | Default | 적용 대상 |
|---|---:|---|
| `ALARM_DISPATCH_MAX_BATCH` | `50` | Valkey/PG consumer drain size |
| `ALARM_DISPATCH_LEASE_SECONDS` | `60` | PG lease |
| `ALARM_DISPATCH_POLL_INTERVAL_MS` | `1000` | PG fallback polling |
| `ALARM_DISPATCH_MAX_BATCHES_PER_WAKE` | `20` | PG fairness yield |
| `ALARM_DISPATCH_RECOVERY_INTERVAL_MS` | `30000` | PG recovery interval |
| `ALARM_DISPATCH_RECOVERY_BATCH_SIZE` | `100` | PG recovery batch |

## Idle waiter env

| Env | Default | 설명 |
|---|---:|---|
| `ALARM_DISPATCH_IDLE_BACKOFF_MIN_MS` | `250` | empty 상태 최초 대기 |
| `ALARM_DISPATCH_IDLE_BACKOFF_MAX_MS` | `5000` | wakeup wait max |
| `ALARM_DISPATCH_POLL_INTERVAL_MS` | `1000` | wakeup unavailable fallback 상한 |

## Retention env

| Env | Default |
|---|---:|
| `ALARM_DISPATCH_RETENTION_ENABLED` | `true` |
| `ALARM_DISPATCH_RETENTION_INTERVAL_MS` | `3600000` |
| `ALARM_DISPATCH_RETENTION_QUERY_TIMEOUT_MS` | `30000` |
| `ALARM_DISPATCH_RETENTION_LIMIT` | `1000` |
| `ALARM_DISPATCH_RETENTION_SENT_DAYS` | `90` |
| `ALARM_DISPATCH_RETENTION_DLQ_DAYS` | `180` |
| `ALARM_DISPATCH_RETENTION_QUARANTINED_DAYS` | `180` |
| `ALARM_DISPATCH_RETENTION_CANCELLED_DAYS` | `90` |
| `ALARM_DISPATCH_RETENTION_EVENT_DAYS` | `90` |

## DB pool env

| Env | Default |
|---|---:|
| `ALARM_WORKER_POSTGRES_POOL_MIN_CONNS` | `1` |
| `ALARM_WORKER_POSTGRES_POOL_MAX_CONNS` | `8` |
| `ALARM_WORKER_POSTGRES_POOL_MAX_IDLE_CONNS` | `4` |
